# Notes about the order of deployment and tests

## Stable Coin Deployment

### 1. Deploy Stable Coin

```bash
npm run deploy:stablecoin
```

### 2. Mint Stable Coin

```bash
npm run mint:stablecoin
```

### 3. Bond Deployment

```bash
npm run deploy:bond
```

### 4. Bond Operations

```bash
npm run ops:bond
```

### 5. Stable Coin Operations

```bash
npm run ops:stable
```
